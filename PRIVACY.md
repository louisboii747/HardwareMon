# HardwareMon Privacy Policy

_Last updated: 30 June 2026_

## Overview

HardwareMon is designed with privacy in mind.

The application performs hardware monitoring, system diagnostics, and benchmarking locally on your device. The current release as if the last updated date does not upload benchmark results, hardware information, or telemetry to HardwareMon servers.

## Information Processed

HardwareMon may process the following information locally in order to provide its features:

- CPU information
- GPU information
- Memory information
- Storage information
- Operating system information
- Live hardware telemetry
- Network statistics
- Process information
- Benchmark results
- Application settings

This information is processed locally and is not transmitted to the developer by the current version of HardwareMon.

## Local Storage

HardwareMon stores certain information on your device, including:

- Application settings
- Historical telemetry (if enabled)
- Benchmark history
- Diagnostic information generated by the user

These files remain on your device unless you choose to remove them.

## Internet Connectivity

Some HardwareMon features may access the internet, for example:

- Checking for application updates
- Downloading new releases when requested by the user

These requests are initiated by the application only for their intended purpose.

## Future Online Features

Future versions of HardwareMon may introduce optional online services, such as anonymous benchmark comparison.

If these features are added:

- Participation will be optional.
- HardwareMon will request your explicit consent before transmitting benchmark data.
- The Privacy Policy will be updated to describe exactly what information is shared.

HardwareMon does not currently upload benchmark results.

## Information That Will Not Be Intentionally Collected for Benchmark Sharing

If anonymous benchmark sharing is introduced in the future, HardwareMon is intended not to collect or transmit personal information such as:

- Usernames
- Personal files
- Installed applications
- Serial numbers
- MAC addresses
- IP addresses (beyond normal network routing required to communicate with the service)

## Open Source

HardwareMon is open source. The source code is publicly available, allowing users to review how the application processes information.

## Changes to This Policy

This Privacy Policy may be updated as HardwareMon evolves. Significant changes to data handling will be reflected in future revisions of this document.
